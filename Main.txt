SUBLIME-BEST Setting FOR LARAVEL & OTHERs

1. Go to the Preferences->setting tab and add the few line of code.
	"font_size": 14,
	"line_padding_top": 12,
	"save_on_focus_lost": true
	
2. Press the CTRL+SHIFT+P and search the Install Package Control & Hit enter.

3. Press again CTRL+SHIFT+P and search the Package Control: Install Package & Hit enter.

4. Repeat the number 2 setting and install these package:
